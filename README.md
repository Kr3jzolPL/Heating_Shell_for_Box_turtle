# Heating-Shell-for-Box-turtle
Controllable heater for box turtle enclosure using your printer with macros for automatic maintaining humidity or drying your filament

| M3 Heated insert | 12       | Heated inserts for fixing stuff.                                                                 |
| M3 Screws | 12       | for Fixing stuff in place.                                                                 |
| PTC Heater 150 watt        | 1        | Main heater 24VDC+24VDC fan                  |
| Thermistor   | 1        | Standard thermistor.                             |
